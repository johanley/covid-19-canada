About the data

  • the time series starts on 2020-03-31
  • updates are made daily, at about 21h ADT (after BC and AB apply their updates)
  • the source data resides on this github repo
  • the data is (almost always) backed by screenshots of government webpages, to let you verify the data
  • for programmers, the data is available as both csv and json files

There are 4 data sets, and 14 jurisdictions (10 provinces, 3 territories, and the federal government):

  • total reported deaths to date (cumulative).
  • total reported cases to date (cumulative). Confirmed cases plus probable/presumptive cases. Only confirmed cases have a positive test result.
  • total completed tests (cumulative). Excludes pending tests. Exception for QC, NL, and NU: this is actually the number of people tested, not the number of tests completed.
  • current reported number of hospitalized patients (non-cumulative).

It's important to note that 3 of the 4 data sets are cumulative, and one is not (hospitalizations).

The federal jurisdiction doesn't report on completed-tests or hospitalizations.
