COVID-19 Canada

This website is currently only for programmers and web designers. It simply provides a JSON API for fetching basic stats related to the COVID-19 epidemic, with a focus on Canada.

The stats are stored as a simple Google spreadsheet, which is updated daily.

The data on the spreadsheets fetched by this website, and made available as a JSON feed. With the data available in that simple form, programmers can more easily create charts and tables related to the epidemic.

This can be extended to other jurisdictions with little effort. As long as the data is in a spreadsheet having the same simple layout, this website is able create a JSON version of that data.

For full documentation, please see the project homepage on github.
